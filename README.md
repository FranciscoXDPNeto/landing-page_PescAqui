# Landing page PescAqui

Projeto para a criação de uma landing page, proposto pelo Professor João Coelho do curso de Engenharia de Software da Univassoras

# Atividade:

A atividade avaliativa, como já informado desde o primeiro dia de aula e reforçado ao longo do semestre será pela entrega e apresentação de Landing page de prática extensionistas. 

Caso o aluno queira apresentar algo a mais, será bem visto, mas o obrigatório é a landing page. Além, deverá terter um formulário que registre o contato e mensagem dos possíveis interessados no projeto.

A primeira versão deverá ser entrega no dia da apresentação. Melhorias podem ser enviadas antes da P2.
